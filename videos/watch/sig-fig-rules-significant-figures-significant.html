Significant figures made easy. Learn the rules for sig figs (significant figures) You may need to know when to count zeros and when not to count zeros.
Here are the rules for sig figs
Count all non-zeros
Count zeros that occur between non-zeros
Leading zeros are not significant
Trailing zeros
If there is a decimal count them
If there is not a decimal don’t count them
