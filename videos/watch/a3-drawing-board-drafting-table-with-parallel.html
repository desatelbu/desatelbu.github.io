Description:



1. A complete set for drawing, with ergonomic design for easy and comfortable working.

2. Robust board of unbreakable, non-warping material for durability.

3. Lubricated guide rails for smooth parallel straightedge movement, with stop-and-go locking mechanism.

4. Rapidly set square and media markings designed for aiding in accuracy.

5. Two clamp strips with central push button operation and corner clamp for easy fixing and checking the media alignment.
