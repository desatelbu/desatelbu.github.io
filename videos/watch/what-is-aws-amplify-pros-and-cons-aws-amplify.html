AWS Amplify is a powerful CLI based tool from AWS. It helps you easily run commands that allow for AWS functionality such as APIs, Databases, User Pools, and much much more.

In this video, I introduce you to some of the concepts of AWS amplify and walk you through some pros and cons of using it. If you’re struggling to decide between Amplify and something else such as SAM, CDK, or Firebase, then this is the video for you.
