I will show you three ways of removing the time from the dates.

First one, copy the data from column A and paste in column B. Select cells in which you want to remove the time

By this method, you can hide the time from the timestamp and you can see only the date now.

2nd Method:
Using the INT formula
Type formula in cell C6 as = INT(A6).

3rd Method:
This is using the Datevalue formula
Type formula in cell D6 as = =DATEVALUE(TEXT(A6,"dd-mmm-yy"))

So like this, you can remove or hide the timestamp from the date value.
