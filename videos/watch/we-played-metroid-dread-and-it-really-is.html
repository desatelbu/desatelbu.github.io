You can forgive long-time Metroid fans going into the first original game since 2010’s somewhat divisive Other M for being a little nervous about the series. But after playing a couple of hours with the opening of #MetroidDread - we have wonderful news. This looks set to be the real deal - a classic 2D Metroid, delivered with some serious production values and all the polish you’d hope for from a game that’s helping launch Nintendo’s latest Switch hardware, next month’s OLED refresh.
