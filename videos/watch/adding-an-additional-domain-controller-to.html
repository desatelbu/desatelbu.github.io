In this ittaster video we begin with a brief overview of the benefits of adding an additional domain controller to a Microsoft Windows Server 2019 domain, followed by a step by step installation and configuration in a learning lab environment.

You’ll learn how to:
1. Join a Windows Server 2019 server to a domain
2. Install the Active Directory Domain Services (AD DS) role
3. Configure an additional Windows Server 2019 domain controller in a test lab environment
4. Check Active Directory objects have successfully replicated using the Active Directory Users & Computers tool
